# Somni logo review set

These files are for review only. I have not updated any app references yet.

Options:
- `option-1-current` keeps the current moon-and-baby feel, but makes the artwork transparent.
- `option-2-redraw` is a cleaner redraw with the same overall idea.

Each option includes:
- `somni-icon.svg` for the standalone mark
- `somni-logo-light.svg` for dark backgrounds
- `somni-logo-dark.svg` for light backgrounds

Open the SVGs directly in the browser to compare them on a transparent background.
