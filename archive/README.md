# Somni Archive

This folder contains completed implementation plans, superseded plans, historical reviews,
handoffs, evaluation artefacts, and old working notes.

Archived files are useful evidence of why earlier decisions were made, but they are not current
instructions. Agents should begin with:

1. `README.md`
2. `docs/Somni_Implementation_Plan_Alpha_1.2.md`
3. `docs/somni_context.md`
4. `docs/somni_architecture.md`

Key completed rollout records include:

- `Implementation_Plan_Alpha_1.1.md`
- `Implementation_Plan_Schedule_Adaptation.md`
- `Implementation_Plan_Notifications.md`
- `Push_Notifications_Task_Record.md`
- `Implementation_Plan_Caregiver_Sharing.md`
- `Implementation_Plan_Support_Dashboard.md`

Do not edit historical records to make them describe the current application. Correct current
behaviour in the live documentation and preserve the archive as a dated snapshot.
